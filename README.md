# Locals App


Android application for adding points of interest to google maps.


## Features

- Add location to map Google Maps API
- Get location data from Google Places API
- Add location to Firebase Database
- Add location to favorite page
- Remove location from favorite page

## Tech Stack

**Client:** Kotlin

**Server:** Kotlin
